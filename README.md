# Social-network

Name: Moment

Description: A social network for sharing moments

## Features

-   Authentication
-   Profile
-   Followers
-   Close Friends
-   Posts
-   Comments
-   Groups
-   Notification
-   Chats

## Frontend

React

### Run instructions for frontend:

Development:

```cmd
    make run dev
```

Docker:

```cmd
    make frontendDocker
```

## Backend

GoLang

### Run instructions for backend:

Development:

```cmd
    make run go
```

Docker:

```cmd
    make backendDocker
```
